# Dedicate

This repository documents how to produce [DCAT](https://www.w3.org/TR/vocab-dcat-3/) metadata for resources documented in [DDI](https://ddialliance.org/). Use cases include:

- DDI-Lifecycle to HealthDCAT-AP

This work is sponsored by [Making Sense](https://making-sense.info/), the [Constances Cohort](https://www.constances.fr/) and the [CASD](https://casd.eu/).
